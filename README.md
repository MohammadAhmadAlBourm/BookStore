# BookStore

## Description

BookStore is a simple web api application built using minimal API in ASP.NET Core. This project demonstrates the usage of minimal API features to create a lightweight and efficient web service for managing a bookstore. With minimal API, the focus is on simplicity and reducing the ceremony typically associated with traditional ASP.NET Core projects.

## Features

- **Minimal API**: Utilizes the minimal API approach in ASP.NET Core for a concise and clean codebase.
- **Book Management**: Provides basic CRUD operations for managing books in the bookstore.
- **Lightweight**: Designed to be lightweight and easy to understand, emphasizing the use of minimal API features.

## Prerequisites

Make sure you have the following installed before running the application:

- [.NET 8 SDK](https://dotnet.microsoft.com/en-us/download/dotnet/8.0)
